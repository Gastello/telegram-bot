# Можливості

-   відстеження змін у Steam Store
-   фільтрація ігор за:
    -   розміром знижки
    -   рейтингом
    -   кількістю відгуків
-   парсинг дати завершення акції
-   генерація кількох варіантів зображення поста
-   модерація перед публікацією
-   підтримка кастомних зображень
-   blacklist ігор
-   запобігання повторним постам

------------------------------------------------------------------------

# Структура файлів

## checker.py

Головний скрипт пошуку нових знижок.

Функції:

-   отримує список змінених ігор зі Steam Store
-   знаходить ігри зі зміною ціни
-   перевіряє:
    -   розмір знижки
    -   рейтинг
    -   кількість відгуків
-   отримує дату завершення знижки
-   надсилає знайдені ігри в модерацію

Запуск:

    python checker.py

Тестовий запуск:

    python checker.py --test-csv test_games.csv

------------------------------------------------------------------------

## moderator_bot.py

Telegram‑бот модерації.

Виконує:

-   обробку команд
-   обробку кнопок модерації
-   прийом кастомних зображень
-   формування фінального preview
-   публікацію поста

Цей скрипт повинен працювати **постійно**.

Запуск:

    python moderator_bot.py

------------------------------------------------------------------------

## bot.py

Модуль відправки постів у модераційний чат.

Виконує:

-   генерацію варіантів зображень
-   відправку варіантів у Telegram
-   додавання кнопок вибору
-   retry при timeout або flood control

------------------------------------------------------------------------

## image_generator.py

Генератор зображень поста.

Функції:

-   отримання фону гри
-   накладення layout
-   відображення:
    -   ціни
    -   старої ціни
    -   дати завершення акції
-   генерація кількох варіантів зображення
-   підтримка кастомного фону

------------------------------------------------------------------------

## steam_store_parser.py

Парсер сторінок Steam.

Виконує:

-   пошук дати завершення знижки
-   аналіз HTML сторінки
-   витягування таймера акції

------------------------------------------------------------------------

## database.py

Робота з SQLite базою.

Зберігає:

-   список вже опрацьованих знижок
-   blacklist
-   moderation items
-   moderation messages
-   кеш перекладів
-   стан синхронізації Steam Store

------------------------------------------------------------------------

## post_formatter.py

Формує текст Telegram поста.

Включає:

-   назву гри
-   нову ціну
-   стару ціну
-   рейтинг
-   кількість відгуків
-   дату завершення знижки

------------------------------------------------------------------------

## config.py

Локальний файл конфігурації.

Створюється вручну.

Приклад:

    BOT_TOKEN = "YOUR_BOT_TOKEN"

    MOD_CHAT_ID = -1000000000000
    CHANNEL_ID = -1000000000000

    STEAM_API_KEY = "YOUR_STEAM_API_KEY"

# Команди бота

/blacklist --- показати blacklist

/block APPID --- заблокувати гру

/unblock APPID --- розблокувати гру

/unblock_last --- розблокувати останню заблоковану

------------------------------------------------------------------------

# Кнопки модерації

Обрати варіант --- вибір зображення

Своє фото --- кастомний background

Post --- публікація

Reject --- відхилення

Block game --- відхилити та заблокувати

# Робочий процес

1.  Запускається moderator_bot.py
2.  Запускається checker.py
3.  Скрипт знаходить знижки
4.  Ігри надсилаються у модерацію
5.  Модератор обирає варіант або завантажує фото
6.  Пост публікується або відхиляється
